
# Assistant

Semgrep Assistant data. Only present if Assistant is enabled

*This model accepts additional fields of type unknown.*

## Structure

`Assistant`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `autofix` | [`Autofix \| undefined`](../../doc/models/autofix.md) | Optional | Fix data generated by Semgrep Assistant |
| `autotriage` | [`Autotriage \| undefined`](../../doc/models/autotriage.md) | Optional | Triage recommendation generated by Semgrep Assistant |
| `component` | [`Component \| undefined`](../../doc/models/component.md) | Optional | Semgrep Assistant's guess as for what the matched source code's purpose is |
| `guidance` | [`Guidance \| undefined`](../../doc/models/guidance.md) | Optional | Remediation guidance generated by Semgrep Assistant |
| `ruleExplanation` | [`RuleExplanation \| undefined`](../../doc/models/rule-explanation.md) | Optional | AI-generated explanation of why a rule flagged this specific code |
| `additionalProperties` | `Record<string, unknown>` | Optional | - |
