
# Autofix

Fix data generated by Semgrep Assistant

*This model accepts additional fields of type unknown.*

## Structure

`Autofix`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `explanation` | `string \| undefined` | Optional | DEPRECATED: This field is deprecated and will always be an empty string. Find a description of how this fix works under `assistant.guidance` |
| `fixCode` | `string \| undefined` | Optional | Source code that replaces all matched lines to fix this finding. AI generated content, review carefully |
| `additionalProperties` | `Record<string, unknown>` | Optional | - |

## Example

```ts
import { Autofix } from 'apimatic-semgrep-sdk';

const autofix: Autofix = {
  explanation: 'explanation2',
  fixCode: 'cookie.setHttpOnly(true);\\nresponse.addCookie(cookie);',
  additionalProperties: {
    'exampleAdditionalProperty': { 'key1': 'val1', 'key2': 'val2' }
  },
};
```
