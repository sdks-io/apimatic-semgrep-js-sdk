
# Autotriage

Triage recommendation generated by Semgrep Assistant

*This model accepts additional fields of type unknown.*

## Structure

`Autotriage`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `reason` | `string \| undefined` | Optional | The reasoning for a `false_positive` verdict; this explains why you might want to ignore the finding. Empty string if verdict is `true_positive` |
| `verdict` | [`Verdict1 \| undefined`](../../doc/models/verdict-1.md) | Optional | The verdict is `true_positive` if Assistant recommends fixing, `false_positive` if Assistant recommends ignoring this finding. AI generated decision, review carefully |
| `additionalProperties` | `Record<string, unknown>` | Optional | - |

## Example

```ts
import { Autotriage, Verdict1 } from 'apimatic-semgrep-sdk';

const autotriage: Autotriage = {
  reason: 'The matched code is used for a non-security related feature.',
  verdict: Verdict1.FalsePositive,
  additionalProperties: {
    'exampleAdditionalProperty': { 'key1': 'val1', 'key2': 'val2' }
  },
};
```
