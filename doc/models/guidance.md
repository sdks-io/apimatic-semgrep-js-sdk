
# Guidance

Remediation guidance generated by Semgrep Assistant

*This model accepts additional fields of type unknown.*

## Structure

`Guidance`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `instructions` | `string \| undefined` | Optional | Step-by-step instructions explaining to a developer how to fix the finding. AI generated content, review carefully |
| `summary` | `string \| undefined` | Optional | Short title explaining to a developer how to fix the finding. AI generated content, review carefully |
| `additionalProperties` | `Record<string, unknown>` | Optional | - |

## Example

```ts
import { Guidance } from 'apimatic-semgrep-sdk';

const guidance: Guidance = {
  instructions: 'instructions4',
  summary: 'Use a template rendering engine such as EJS instead of string concatenation.',
  additionalProperties: {
    'exampleAdditionalProperty': { 'key1': 'val1', 'key2': 'val2' }
  },
};
```
